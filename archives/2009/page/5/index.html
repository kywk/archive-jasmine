[ 初夏的北海道(先行版) ] 尼尼爾潘

這禮拜Jasmine要忙著搬家~ 大家不要再當催狂魔了QQQQQQQQQQ



讓我偷懶慢慢修圖吧QQ



先偷尼尼先生的圖來貼貼 呼呼呼 XDDD




以下轉貼自尼尼爾潘的網誌





2009年3月，開始著手企畫北海道之旅，並且選定在端午連假出發



距離上次離開到別的國家，已經是接近十五年前的事，這也是第一次企畫出國自助旅行



列了很多點，有很多雄心壯志，想要在扣掉飛行時間後的短短三天，造訪很多地方



到後來才發現，這不是一個拍什麼人像攝影作品之旅，而是一個純粹的旅行，去實踐很久以前就想去北海道的夢想



去真正踏上那個高緯度，即使夏天晚上也只有十幾度



卻擁有寬闊的天空、無止境的山丘、充滿設計感卻又很溫馨的小房子、以及迷人古老建築的地方



當然，經過每天耗盡體力的行程，一碗熱呼呼，叉燒跟臉一樣大的拉麵也絕對不容錯過



除了每天上下班的尖峰時刻，除了札幌薄野區每晚的夜生活，此行是極度安靜的、平靜的，悠閒已不足以形容



北海道就是這樣的一個地方



遊記希望能慢慢寫，先來幾張圖分享 : )
