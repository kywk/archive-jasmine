[ Salvation code 3.1 ]

生還者: 1名……….Jasmine

救援部隊: 3名……..ANNY RAKKI NEILPAN

任務: 打擊活死人拯救全人類(大誤)





呼…千呼萬喚使出來XDD

Jasmine難得有殺氣這麼重的作品

起因完全只是因為一個在士林夜市敗家的槍套(誤)

是因為很久很久之前 Jasmine就很想要以羅拉的形象出來嚇人(再誤!)

這次的拍攝計畫其實是臨時起意XD

跟著要去十三層探險的米大和克里克一行人一起去大冒險



前一天晚上在準備道具的時候

笨蛋Jasmine才忽然發現自己沒有準備軍火阿阿阿阿阿 囧

情急之下 我只好使出了大絕招: CALL OUT 親友團= =+

陽光活力美少女 安妮公主 以及 英勇強壯肌肉猛男 RAKKI

義無反顧的帶著彈藥來救援我們這次的任務

安妮公主的技術指導真的專業到一個不行

RAKKI供應的軍火也是火力爆炸強大的阿>O<

如果不是你們的友情 就不會有這麼殺的作品出現了 嗚嗚

這就是地球人的愛啊!(淚光閃閃)

所以阿

Salvation code 3.1 的真正意涵

就是ANNY RAKKI NEIL三個人 指點我一個人^__^

這是大家一起合作的結晶喔~



下次的計畫就是霹靂雙嬌了= =+

大家可以期待安妮公主的超正表現唷唷>////<(啾)
